---
title: "Create Quote"
openapi: "POST /organizations/{organization_id}/quotes"
description: "Creates a quote for an organization"
---

Creates a quote for an organization.

## Common Headers

| Header          | Value              | Required | Description                     |
| :-------------- | :----------------- | :------- | :------------------------------ |
| `Authorization` | `Bearer <api-key>` | true     | Your API key for authentication |
| `Content-Type`  | `application/json` | true     | Content type of the request     |

## Path

<ParamField path="organization_id" type="string" required>
  ID of organization
</ParamField>

## Body

<ParamField body="offering_id" type="string" required>
  ID of the offering to quote
</ParamField>

<ParamField body="payin_amount" type="string">
  Amount to pay in (specify either payin_amount or payout_amount, not both)
</ParamField>

<ParamField body="payout_amount" type="string">
  Amount to receive as payout (specify either payin_amount or payout_amount, not
  both)
</ParamField>

<Warning>
  Either `payin_amount` or `payout_amount` may be specified but not both.
</Warning>
