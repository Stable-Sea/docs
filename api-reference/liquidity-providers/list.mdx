---
title: "List Liquidity Providers"
openapi: "GET /liquidity-providers"
description: "Returns all available liquidity providers with their trading pairs and hours of operation"
---

Returns all available liquidity providers with their supported trading pairs and operating schedules.

## Common Headers

| Header          | Value              | Required | Description                     |
| :-------------- | :----------------- | :------- | :------------------------------ |
| `Authorization` | `Bearer <api-key>` | true     | Your API key for authentication |
| `Content-Type`  | `application/json` | true     | Content type of the request     |
