---
title: "List Hours of Operation"
openapi: "GET /hours-of-operation"
description: "Returns hours of operation for liquidity providers"
---

Returns hours of operation for liquidity providers.

## Common Headers

| Header          | Value              | Required | Description                     |
| :-------------- | :----------------- | :------- | :------------------------------ |
| `Authorization` | `Bearer <api-key>` | true     | Your API key for authentication |
| `Content-Type`  | `application/json` | true     | Content type of the request     |

## Response

<ResponseExample>

```json Response
{
  "data": [
    {
      "liquidity_provider": "ALPHA",
      "payin_currency": "USDC",
      "payout_currency": "COP",
      "hours_of_operation": [
        {
          "open_time_utc": "2024-01-15T09:00:00Z",
          "close_time_utc": "2024-01-15T17:00:00Z"
        }
      ]
    }
  ]
}
```

</ResponseExample>
