---
title: "Create Customer"
openapi: "POST /customers"
description: "Creates a new customer"
---

Creates a new customer in the system.

## Common Headers

| Header          | Value              | Required | Description                     |
| :-------------- | :----------------- | :------- | :------------------------------ |
| `Authorization` | `Bearer <api-key>` | true     | Your API key for authentication |
| `Content-Type`  | `application/json` | true     | Content type of the request     |

## Body

<ParamField body="name" type="string" required>
  The name of the customer
</ParamField>

<ParamField body="contact" type="object" required>
  Contact information for the customer
</ParamField>

<ParamField body="contact.email" type="string" required>
  Email address of the customer contact
</ParamField>

<ParamField body="contact.first_name" type="string" required>
  First name of the customer contact
</ParamField>

<ParamField body="contact.last_name" type="string" required>
  Last name of the customer contact
</ParamField>
