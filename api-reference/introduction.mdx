## Welcome to the Terminal API

The Stable Sea Terminal API provides programmatic access to terminal operations, enabling you to integrate shipping and logistics services directly into your applications. This RESTful API supports the complete workflow from organization setup to order fulfillment.

## Base URL

All API requests should be made to:

```
https://api.stablesea.com/terminal/v1
```

## Authentication

The Terminal API uses API key authentication. Include your API key in the Authorization header of all requests:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Getting Started

Follow this workflow to get started with the Terminal API:

1. **Authenticate** - Obtain and configure your API key
2. **Create Organization** - Set up your organization profile
3. **Request Quote** - Get pricing for your terminal services
4. **Place Order** - Submit your service request

## Rate Limits

API requests are limited to ensure service quality:

- **Standard tier**: 1,000 requests per hour
- **Premium tier**: 10,000 requests per hour

Rate limit headers are included in all responses to help you manage your usage.
