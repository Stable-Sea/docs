---
title: "Create Organization"
openapi: "POST /organizations"
description: "Creates a new organization"
---

Creates a new organization in the system.

## Common Headers

| Header            | Value              | Required | Description                                |
| :---------------- | :----------------- | :------- | :----------------------------------------- |
| `Authorization`   | `Bearer <api-key>` | true     | Your API key for authentication            |
| `Content-Type`    | `application/json` | true     | Content type of the request                |
| `Idempotency-Key` | `<unique-uuid>`    | true     | Unique key to prevent duplicate operations |

## Body

<ParamField body="name" type="string" required>
  The name of the organization
</ParamField>

<ParamField body="contact" type="object" required>
  Contact information for the organization
</ParamField>

<ParamField body="contact.email" type="string" required>
  Email address of the organization contact
</ParamField>

<ParamField body="contact.first_name" type="string" required>
  First name of the organization contact
</ParamField>

<ParamField body="contact.last_name" type="string" required>
  Last name of the organization contact
</ParamField>

## Code examples

<CodeGroup>
  ```bash curl
  curl -X POST "https://api-sandbox.stablesea.com/v1/organizations" \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -H "Idempotency-Key: $(uuidgen)" \
    -d '{"name": "Acme Corp", "contact": {"email": "john@acme.com", "first_name": "John", "last_name": "Doe"}}'
  ```
  ```go Go
  package main

  import (
      "context"
      "github.com/google/uuid"
      "github.com/stablesea/sdk-go/api"
      "github.com/stablesea/sdk-go"
  )

  func main() {
      client := sdk.NewClient("YOUR_API_KEY", "")
      org, err := client.CreateOrganization(context.Background(), uuid.New().String(), api.NewOrganization{
          Name: "Acme Corp",
          Contact: api.Contact{Email: "john@acme.com", FirstName: "John", LastName: "Doe"},
      })
      if err != nil {
          panic(err)
      }
      _ = org
  }
  ```
  ```typescript TypeScript
  import { createClient } from '@stablesea/sdk';

  const client = createClient({ apiKey: 'YOUR_API_KEY' });
  const org = await client.organizations.create(
    { name: 'Acme Corp', contact: { email: 'john@acme.com', first_name: 'John', last_name: 'Doe' } },
    'unique-idempotency-key-123'
  );
  ```
  ```python Python
  from stablesea import StableseaClient
  import uuid

  client = StableseaClient(api_key="YOUR_API_KEY")
  org = client.organizations.create(
      name="Acme Corp",
      contact={"email": "john@acme.com", "first_name": "John", "last_name": "Doe"},
      idempotency_key=str(uuid.uuid4()),
  )
  client.close()
  ```
