---
title: "Get Organization"
openapi: "GET /organizations/{organization_id}"
description: "Retrieve details for a specific organization"
---

Retrieve details for a specific organization.

## Common Headers

| Header          | Value              | Required | Description                     |
| :-------------- | :----------------- | :------- | :------------------------------ |
| `Authorization` | `Bearer <api-key>` | true     | Your API key for authentication |
| `Content-Type`  | `application/json` | true     | Content type of the request     |

## Path

<ParamField path="organization_id" type="string" required>
  ID of organization to retrieve
</ParamField>
