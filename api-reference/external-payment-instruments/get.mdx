---
title: "Get External Payment Instrument"
openapi: "GET /organizations/{organization_id}/external_payment_instruments/{external_payment_instrument_id}"
description: "Fetch a specific external payment instrument for a organization"
---

This endpoint is used to fetch a specific external payment instrument for a organization.

## Common Headers

| Header          | Value              | Required | Description                     |
| :-------------- | :----------------- | :------- | :------------------------------ |
| `Authorization` | `Bearer <api-key>` | true     | Your API key for authentication |
| `Content-Type`  | `application/json` | true     | Content type of the request     |

## Path

<ParamField path="organization_id" type="string" required>
  ID of organization
</ParamField>

<ParamField path="external_payment_instrument_id" type="string" required>
  ID of external payment instrument
</ParamField>
