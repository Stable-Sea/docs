---
title: "Delete Organization"
openapi: "DELETE /organizations/{id}"
---

Delete an organization from the Terminal system. This action is irreversible.

## Example Response

```json
{
  "message": "Organization deleted successfully",
  "id": "org_12345",
  "deleted_at": "2024-01-15T10:30:00Z"
}
```

<Warning>
  Deleting an organization will permanently remove all associated data and
  cannot be undone.
</Warning>

<Note>This is dummy data for demonstration purposes.</Note>
