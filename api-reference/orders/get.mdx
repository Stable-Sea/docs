---
title: "Get Order"
openapi: "GET /organizations/{organization_id}/orders/{order_id}"
description: "Retrieve details for a specific order"
---

Retrieve details for a specific order.

## Common Headers

| Header          | Value              | Required | Description                     |
| :-------------- | :----------------- | :------- | :------------------------------ |
| `Authorization` | `Bearer <api-key>` | true     | Your API key for authentication |
| `Content-Type`  | `application/json` | true     | Content type of the request     |

## Path

<ParamField path="organization_id" type="string" required>
  ID of organization
</ParamField>

<ParamField path="order_id" type="string" required>
  ID of order
</ParamField>
