---
title: "Create Order"
openapi: "POST /organizations/{organization_id}/orders"
description: "Creates an order for an organization"
---

Creates an order for an organization based on a quote.

## Common Headers

| Header            | Value              | Required | Description                                |
| :---------------- | :----------------- | :------- | :----------------------------------------- |
| `Authorization`   | `Bearer <api-key>` | true     | Your API key for authentication            |
| `Content-Type`    | `application/json` | true     | Content type of the request                |
| `Idempotency-Key` | `<unique-uuid>`    | true     | Unique key to prevent duplicate operations |

## Path

<ParamField path="organization_id" type="string" required>
  ID of organization
</ParamField>

## Body

<ParamField body="quote_id" type="string" required>
  ID of the quote to create an order from
</ParamField>
